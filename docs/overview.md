# Prompt Improver

## Purpose

A web application designed to help junior developers in the Israeli high-tech industry improve their AI prompting skills. Users submit prompts and receive detailed feedback on strengths and weaknesses.

## Target Audience

- Junior developers entering the Israeli tech workforce
- Bootcamp graduates transitioning to professional roles
- Students learning to work with AI tools effectively

## Core Features

### MVP Features
- Simple chat-like interface (RTL support for Hebrew)
- Prompt input field
- Hebrew language UI
- Detailed feedback display showing:
  - Pros (what works well in the prompt)
  - Cons (areas for improvement)
  - Suggestions for better prompting

### Future Considerations
- Prompt history
- Example prompts library
- English language support
- Scoring system
